## API gotchas

- Algolia uses API_KEY auth mode with `X-Algolia-API-Key: API_KEY`and `X-Algolia-Application-Id: APP_ID` in the request header to access different endpoints.

<Note>Contribute API gotchas by [editing this page](https://github.com/nangohq/nango/tree/master/docs-v2/integrations/all/algolia.mdx)</Note>
