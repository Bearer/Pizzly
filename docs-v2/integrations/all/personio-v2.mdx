## API gotchas
    - If you want to use the recruiting API, you should use the `personio` provider. This `v2` is only for the main Personio API.
    - An example of setting them in the `nango.auth` call:
    ```ts
    nango.auth('personio', 'test-connection-id', { params: { partnerId: 'YOUR_CUSTOM_NAME', appId: 'UNIQUE_NAME' },
        credentials: {
            client_id: '<your-client-id>',
            client_secret: '<your-client-secret>'
        }
      })
      .then((result: { providerConfigKey: string; connectionId: string }) => {
        // do something
      }).catch((err: { message: string; type: string }) => {
        // handle error
      });
    ```
