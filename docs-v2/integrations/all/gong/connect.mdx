# Overview

To authenticate with Gong, you need two key piece of information:
1. **Access Key** - Your Gong Access Key Key
2. **Access Key Secret** - Your Gong Access Key Secret


This guide will walk you through finding or creating those credentials within Gong.


### Prerequisites:

- You must set up an account with Gong.


#### Step 1: Finding Gong API Key and API Key secret
1. Log into your Gong account and navigate to `Company Settings` > `Ecosystem` > `API` and click the `Create`
to generate the access key and the access key secret.


2. Securely store you access key and the access key secret.

#### Step 2: Connect Your Gong Account


To authenticate using your API credentials, navigate to the Gong authentication form.
Enter your credentials as follows:
- username: Your Gong Access Key
- password: Your Gong Access Key Secret

Click "Connect" to complete the authentication.
