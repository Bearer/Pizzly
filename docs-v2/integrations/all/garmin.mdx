## Getting started

- Garmin's API documentation is private, you must create a developer account at [Garmin Developer Portal](https://developerportal.garmin.com) to access them
- Garmin uses the Oauth1(a) authorization protocol as outlined [here](https://oauth.net/core/1.0a/)

<Tip>Need help getting started? Get help in the [community](https://nango.dev/slack).</Tip>
