## API gotchas

-   While setting up the OAuth app, use the `https://www.googleapis.com/auth/documents` scope for extended capabilities.
-   To enable `Google Docs` in the developer console for your project, please use the following link: https://console.cloud.google.com/flows/enableapi?apiid=docs.googleapis.com
