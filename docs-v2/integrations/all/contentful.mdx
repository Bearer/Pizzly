## API gotchas
-   Contentful also supports the authorization code flow as well.
-   Contentful requires specific subdomain for the various API type this will comprise of the API type and region. i.e `cdn.eu` for the Content Delivery API EU region, `cdn` for the Content Delivery API, or `api.eu` for Content Management API EU region. For more details on this please check the introduction section of each API type.
-   Different API types enforce different rate limits, for more details on this check the API rate limits section for the various API types. i.e [Content delivery API rate limit](https://www.contentful.com/developers/docs/references/content-delivery-api/#/introduction/api-rate-limits) for Content Delivery API.
