## Setup guide

<Steps>
  <Step title="Login/Sign-up to Google AI Studio">
    Go to [Google AI Studio](https://aistudio.google.com) and login or sign up using your Google account.
  </Step>

  <Step title="Generate Gemini API key">
    Click _Get API key_ button. This will display a list of your existing API keys. If you don't have one, create a new key. Once generated, copy the API key—you'll need to provide it in Nango when setting up your Gemini connection.
  </Step>

  <Step title="Next">
    Follow the [_Quickstart_](/getting-started/quickstart).
  </Step>
</Steps>

## API gotchas

- Keeping your Gemini API key secure is essential. Unauthorized access allows others to use your project's quota, potentially resulting in quota loss or unexpected charges for billed projects. It also exposes your tuned models and files to unauthorized use.
