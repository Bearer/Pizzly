# Overview

To authenticate with Calendly, you need this key pieces of information:
1. **API Token** - A key that grants full access to all your application and configurations.


### Prerequisites:

- You must set up an account with Calendly.


#### Step 1: Finding API Token
1. Log into your calendly account and nagivate to **Integation & apps** and click on **API and webhooks**

<img src="/integrations/all/calendly/nango_calendly.png" />

1. Under the header **Personal access tokens** click `Get a token now` to create an API Token

<img src="/integrations/all/calendly/nango_calendly_create_token.png" />

1. After creating the token, a modal with the API Token is visbile. Securely copy and store the API token.

<img src="/integrations/all/calendly/nango_calendly_token.png" />



#### Step 2: Creation calendly connection in Connect UI

Note: Do not disable pop ups.

1. Once in the Connect UI, click the **connect** button. Do not close the dialog. 

<img src="/integrations/all/calendly/calendly_connect_slim" />

1. After a few seconds, you should be successfully authenticated.

You are now connected to Calendly.

<img src="/integrations/all/calendly/calendly_sucess.png" />
