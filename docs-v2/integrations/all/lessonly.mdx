## API gotchas

- Lessonly uses `Basic` auth mode to access different endpoints. Provide your `SUBDOMAIN:API_KEY` as the username value and leave the password field empty.
- For rate limit details, check out [Lessonly rate limiting](https://docs.lessonly.com/#rate-limiting).
