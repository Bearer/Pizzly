<Tip>Need help getting started? Get help in the [community](https://nango.dev/slack).</Tip>

## API gotchas

- Chattermill uses API_KEY auth mode with `Authorization: Bearer apiKey` in the request header to access different endpoints.
- When creating a new connection on Nango, you can specify the subdomain for your request, which varies for different resources within the Chattermill API.
