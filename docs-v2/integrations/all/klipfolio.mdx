## API gotchas

- Klipfolio uses API_KEY auth mode with `kf-api-key: API_KEY` as a request header to access different endpoints.
- Klipfolio enforces rate limits for its endpoints based on the subscribed plan. For more details check [Klipfolio rate limits](https://apidocs.klipfolio.com/reference/getting-started#rate-limiting)
