
## Pre-built use cases

<AccordionGroup>


<Accordion title="Files">
| Endpoint | Description | Readme |
| - | - | - |
| `PUT /files` | Write content to a particular github file within a repo. If<br />the file doesn't exist it creates and then writes to it | [github.com](https://github.com/NangoHQ/integration-templates/blob/main/integrations/github/actions/write-file.md) |
| `GET /files` | Lists all the files of a Github repo given a specific branch | [github.com](https://github.com/NangoHQ/integration-templates/blob/main/integrations/github/actions/list-files.md) |
</Accordion>


<Accordion title="Others">
| Endpoint | Description | Readme |
| - | - | - |
| `GET /github/list-repos` | List github repos from an organization. | [github.com](https://github.com/NangoHQ/integration-templates/blob/main/integrations/github/actions/list-repos.md) |
| `POST /github/create-demo-issue` | Create a GitHub issue in Nango's showcase repository. | [github.com](https://github.com/NangoHQ/integration-templates/blob/main/integrations/github/actions/create-demo-issue.md) |
| `GET /github/issues` | Fetches Github issues from all a user's repositories | [github.com](https://github.com/NangoHQ/integration-templates/blob/main/integrations/github/actions/issues.md) |
| `GET /github/issues-lite` | Fetches Github issues but up to a maximum of 15 | [github.com](https://github.com/NangoHQ/integration-templates/blob/main/integrations/github/actions/issues-lite.md) |
| `GET /github/demo-issues` | Fetches GitHub issues from our showcase repository. | [github.com](https://github.com/NangoHQ/integration-templates/blob/main/integrations/github/actions/issues-demo.md) |
</Accordion>
