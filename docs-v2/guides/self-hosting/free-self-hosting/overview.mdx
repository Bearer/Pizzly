All free feature available on Nango Cloud are also available on the Free self-hosted edition.

This includes: 
- Authorization for 300+ APIs
- All authorization mode (OAuth, API key, basic, custom)
- Fully white-label authorization flow
- Optional pre-built UI for the authorization flow
- Authorization input format validation 
- End-user guides
- Invalid credentials detection
- Management dashboard
- Logs & monitoring
- Encryption at rest
- Programmatic API & SDKs
