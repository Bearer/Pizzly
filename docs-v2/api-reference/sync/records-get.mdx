### The `delta` parameter & webhooks

If you are using the [Nango Sync webhooks](/nango-sync/guides/webhooks#from-nango) you can use the timestamp you get from the webhook for the `delta` parameter.

### Response

Returns an array of objects in the data model that you queried.

For example:
```json
[
    {
        id: 123,
        // Fields as specified in the model you queried
        ...
    },
    ...
]
```
